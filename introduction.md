# 1、绪论

### 一、什么叫电机

是利用电磁感应原理工作的机械。

#### 电机的分类

| 能量转化                         | 学科不同                     | 应用场合要求和电源 |
| -------------------------------- | ---------------------------- | ------------------ |
| 发电机、电动机、变压器、控制电机 | 静止电机（变压器）  旋转电机 | 直流电机、交流电机 |

#### 电机的作用

* 电能的生产、传输、分配的主要设备
* 动力设备
* 控制系统重要原件

### 二、电力拖动于电气控制研究的内容

电力拖动系统时用电动机来拖动机械运行的系统。
